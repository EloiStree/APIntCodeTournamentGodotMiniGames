# 2025_11_20_CodeTournamentGodotMiniGames
Project in Godot to showcase the architecture of multiplayer mini games to learn code from.

- You need:
  - a Websocket to push out game state information as bytes and texts.
    - also the manual of the game RTFM_START_MD ... RTFM_STOP_MD
  - a websocket to push in the game input
  - a parser that turn integer IID to keyboard and gamepad controller.
    - a 1899999999 joystick receiver
    - a 1787654321 Xbox buttons receiver
    - a 1000 2000 S2W receiver for keyboard
  - a ntp connection to a server
  - a IID delayer for date information
